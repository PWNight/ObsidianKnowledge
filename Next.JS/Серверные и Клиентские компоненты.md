Используйте клиентские компоненты, когда вам нужно:
* Использовать [[Состояние (State)]] и [[Обработчик Ивентов (Event Handler)]], например onClick, onChange и т.п.
* Использовать [[Жизненный цикл]], например useEffect и т.п.
* Работать с браузерными API по типу `localStorage`, `window`, `Navigator.geolocation` и т.п.
* Использовать [[Собственные хуки]]

Используйте серверные компоненты, когда вам нужно:
* Получать информацию из БД или АПИ.
* Использовать АПИ ключи, токены и прочие секреты, которые должны быть изолированы от клиента.
* Уменьшить количество отправляемого JS на браузер.

Например. Серверный компонент - это компонент, который получает информацию о посте и передаёт их на клиент, который дальше обрабатывает и отображает эту информацию.

```tsx
import LikeButton from '@/app/ui/like-button'
import { getPost } from '@/lib/data'
 
export default async function Page({
  params,
}: {
  params: Promise<{ id: string }>
}) {
  const { id } = await params
  const post = await getPost(id)
 
  return (
    <div>
      <main>
        <h1>{post.title}</h1>
        {/* ... */}
        <LikeButton likes={post.likes} />
      </main>
    </div>
  )
}
```

Пример клиентского компонента:
```tsx
'use client'
 
import { useState } from 'react'
 
export default function Counter() {
  const [count, setCount] = useState(0)
 
  return (
    <div>
      <p>{count} likes</p>
      <button onClick={() => setCount(count + 1)}>Click me</button>
    </div>
  )
}
```

Также компоненты можно импортировать из различных библиотек и дальше использовать их, как собственные.
```tsx
import Carousel from './carousel'
 
export default function Page() {
  return (
    <div>
      <p>View pictures</p>
      <Carousel />
    </div>
  )
}
```
