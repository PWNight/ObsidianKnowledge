Файлы верхнего уровня используются для настройки приложения, управления зависимостями, запуска промежуточного программного обеспечения, интеграции инструментов мониторинга и определения переменных среды.

| Файл                                                                                           | Описание                                              |
| ---------------------------------------------------------------------------------------------- | ----------------------------------------------------- |
| [`next.config.js`](https://nextjs.org/docs/app/api-reference/config/next-config-js)            | Файл конфигурации для Next.js                         |
| [`package.json`](https://nextjs.org/docs/app/getting-started/installation#manual-installation) | Зависимости проекта и скрипты                         |
| [`instrumentation.ts`](https://nextjs.org/docs/app/guides/instrumentation)                     | Файл OpenTelemetry и Instrumentation                  |
| [`middleware.ts`](https://nextjs.org/docs/app/api-reference/file-conventions/middleware)       | Промежуточное программное обеспечение запроса Next.js |
| [`.env`](https://nextjs.org/docs/app/guides/environment-variables)                             | Переменные среды                                      |
| [`.env.local`](https://nextjs.org/docs/app/guides/environment-variables)                       | Локальные переменные среды                            |
| [`.env.production`](https://nextjs.org/docs/app/guides/environment-variables)                  | Переменные производственной среды                     |
| [`.env.development`](https://nextjs.org/docs/app/guides/environment-variables)                 | Переменные среды разработки                           |
| [`.eslintrc.json`](https://nextjs.org/docs/app/api-reference/config/eslint)                    | Файл конфигурации для ESLint                          |
| `.gitignore`                                                                                   | Файлы и папки Git, которые следует игнорировать       |
| `next-env.d.ts`                                                                                | Файл объявления TypeScript для Next.js                |
| `tsconfig.json`                                                                                | Файл конфигурации для TypeScript                      |
| `jsconfig.json`                                                                                | Файл конфигурации для JavaScript                      |
