
| Команда     | Описание                                                                                                                                                                |
| ----------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| pwd         | Выводит текущую папку                                                                                                                                                   |
| cd          | Переходит в папку:<br>1. `cd ..` переходит в предыдущую папку<br>2. `cd ~` переходит в корневую папку                                                                   |
| history     | Показывает историю команд терминала                                                                                                                                     |
| ls          | Показывает содержимое текущей папки:<br>1. `ls -a` показывает скрытое содержимое<br>2. `ls -l` показывает содержимое в виде таблицы <br>с полезной инфой<br>3. `ls -la` |
| mkdir       | Создаёт папку с указанным названием                                                                                                                                     |
| tree        | Выводит содержимое папки в виде дерева                                                                                                                                  |
| touch       | Создаёт файл                                                                                                                                                            |
| \> filename | Создаёт файл                                                                                                                                                            |
| echo        | Создаёт файл                                                                                                                                                            |
| cat         | Создаёт файл                                                                                                                                                            |
