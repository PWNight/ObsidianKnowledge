### **Навигация по строке ввода**

| Сочетание клавиш          | Действие                                                            |
| ------------------------- | ------------------------------------------------------------------- |
| **Ctrl + A**              | Переместить курсор в начало строки                                  |
| **Ctrl + E**              | Переместить курсор в конец строки                                   |
| **Ctrl + F**              | Переместить курсор на один символ вперёд                            |
| **Ctrl + B**              | Переместить курсор на один символ назад                             |
| **Ctrl + XX**             | Переключить позицию курсора между началом строки и текущей позицией |
| **Ctrl + ] + x**          | Переместить курсор вперёд к следующему вхождению символа `x`        |
| **Alt + F** / **Esc + F** | Переместить курсор на одно слово вперёд                             |
| **Alt + B** / **Esc + B** | Переместить курсор на одно слово назад                              |
| **Alt + Ctrl + ] + x**    | Переместить курсор к предыдущему вхождению символа `x`              |

### **Управление терминалом и процессами**

|Сочетание клавиш|Действие|
|---|---|
|**Ctrl + L**|Очистить экран терминала (аналогично команде `clear`)|
|**Ctrl + S**|Приостановить вывод команды на экран|
|**Ctrl + Q**|Возобновить вывод, приостановленный **Ctrl+S**|
|**Ctrl + Z**|Приостановить выполнение текущей команды и отправить её в фон|
|**Ctrl + C**|Отправить сигнал SIGINT и завершить выполняющуюся команду|
|**Ctrl + D**|Закрыть текущий терминал (отправить EOF)|

### **Работа с историей команд**

|Сочетание клавиш|Действие|
|---|---|
|**Ctrl + R**|Инкрементный обратный поиск по истории|
|**Alt + P**|Неинкрементный обратный поиск по истории|
|**Ctrl + J**|Завершить поиск по истории, оставив текущую команду|
|**Ctrl + G**|Выйти из режима поиска по истории|
|**Ctrl + _**|Отменить последнее изменение в строке|
|**Ctrl + P** / **Стрелка вверх**|Перейти к предыдущей команде в истории|
|**Ctrl + N** / **Стрелка вниз**|Перейти к следующей команде в истории|
|**Ctrl + S** (в поиске)|Перейти к следующей (более старой) команде в результатах поиска|
|**Ctrl + O**|Выполнить найденную команду (через **Ctrl+R/S**) и продолжить поиск|

### **Специальные команды истории (вводятся в строке)**

|Команда|Действие|
|---|---|
|**!!**|Выполнить последнюю команду|
|**!***|Выполнить предыдущую команду, кроме первого слова|
|**!*:p**|Показать, что подставит команда `!*` (без выполнения)|
|**!x**|Выполнить последнюю команду из истории, начинающуюся с `x`|
|**!x:p**|Показать команду, начинающуюся с `x`, и добавить её в историю|
|**!$** / **Alt + .**|Подставить последний аргумент предыдущей команды|
|**!$:p**|Показать, что подставит `!$` (без выполнения)|
|**!^**|Подставить первый аргумент предыдущей команды|
|**^123^abc**|Заменить в предыдущей команде `123` на `abc` и выполнить|
|**!n**|Выполнить команду под номером `n` из истории|
|**!n:p**|Показать команду под номером `n` (без выполнения)|
|**!n:m**|Повторить аргументы с номера `n` по `m` из последней команды|
|**!n:$**|Повторить аргументы от `n` до последнего из последней команды|
|**!fi**|Выполнить последнюю команду из истории, начинающуюся с `fi`|

### **Редактирование строки ввода**

|Сочетание клавиш|Действие|
|---|---|
|**Ctrl + U**|Удалить от курсора до начала строки|
|**Ctrl + K**|Удалить от курсора до конца строки|
|**Ctrl + W**|Удалить слово/аргумент перед курсором|
|**Ctrl + D**|Удалить символ под курсором (или закрыть терминал)|
|**Ctrl + H**|Удалить символ перед курсором (как Backspace)|
|**Alt + D**|Удалить от курсора до конца слова|
|**Alt + Backspace**|Удалить от курсора до начала слова|
|**Alt + .** / **Esc + .**|Вставить последний аргумент предыдущей команды|
|**Alt + <**|Перейти к первой команде в истории|
|**Alt + >**|Перейти к последней команде в истории|
|**Esc + T**|Поменять местами два последних слова перед курсором|
|**Alt + T**|Поменять местами текущее и предыдущее слово|

### **Автодополнение и информация**

|Сочетание клавиш|Действие|
|---|---|
|**Tab**|Автодополнение команды или имени файла/каталога|
|**Ctrl + I**|Аналогично **Tab** (автодополнение)|
|**~** затем **Tab Tab**|Показать список всех пользователей системы|
|**Alt + ?**|Показать список файлов и каталогов в текущем пути (справка)|
|**Alt + ***|Вставить имена всех файлов и каталогов в текущем пути как аргументы|
